Grid systems

Hippo Theme is based on Bootstrap and thus supports the default Bootstrap grid and its breakpoints.

Short summary of Bootstrap grid options:

Grid behavior Horizontal at all times Collapsed to start, horizontal above breakpoints
Container width None (auto) 750px 970px 1170px
Class prefix .col-xs- .col-sm- .col-md- .col-lg-
# of columns 12

Gutter width 30px (15px on each side of a column)

Flexible pane grid, full height columns / full width rows

Based on the idea of the new flexbox css module, see a full explanation here.

This grid will let you stretched panes in either vertical (column) or horizontal (row) fashion in a 12 pane grid. It is a flexible grid, meaning that it uses a percentage of the viewport to scale the panes. The area of a pane is defined by giving it a class with area boundaries, for example .pane-xs-0-6 will make the pane stretch from beginning to half of 12 pane grid.

This also allows you to reorder the columns as you could put a .pane-xs-4-6 before a .pane-xs-2-4 in your html but it will visually still show up after the 2-4 pane.

Just like the bootstrap theme the grid has breakpoint variations allowing for different layouts per breakpoint.

The parent of the pane, or the .pane-row or .pane-column does need its height set explicitly as absolute positioned elements are not part of the normal page flow.

Try it out! Resize your viewport and see the below grids change and switch.

There is also a less mixin available .pane(@top, @right, @bottom, @left) which allows to make any element positioned absolute and stretched to the dimentions specified.
