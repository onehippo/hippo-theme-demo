Grid systems

Hippo Theme is based on Bootstrap and thus supports the default Bootstrap grid and its breakpoints.

Flexible pane grid with full height columns and full width rows

Based on the idea of the new flexbox css module, see a full explanation here.

A pane grid can be used as a 12 column grid with full height, or a 12 row grid with full width. It is also flexible, as it uses a percentage of the viewport to scale the panes.

The area of a pane is defined by giving it a class with the start and end row or column. For example, in a .pane-row a .pane-xs-1-6 will make the pane stretch from column 1 to 6 (i.e. half of the 12 column grid).

This also allows you to reorder the panes. A .pane-xs-5-6 can be put before a .pane-xs-2-4 but it will visually still show up after the 2-4 pane.

Just like the bootstrap theme the grid has breakpoint variations allowing for different layouts per breakpoint. To collapse a pane (i.e. hide it) from a specific breakpoint, use the class .pane-{breakpoint}-collapse.

The parent of the pane, or the .pane-row or .pane-column does need its height set explicitly as absolute positioned elements are not part of the normal page flow.
