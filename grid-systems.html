Grid systems

Hippo Theme is based on Bootstrap and thus supports the default Bootstrap grid and its breakpoints.

Short summary of Bootstrap grid options:

Extra small devices Phones (<768px) Small devices Tablets (≥768px) Medium devices Desktops (≥992px) Large devices Desktops (≥1200px)
Grid behavior Horizontal at all times Collapsed to start, horizontal above breakpoints
Container width None (auto) 750px 970px 1170px
Class prefix .col-xs- .col-sm- .col-md- .col-lg-
# of columns 12
Column width Auto ~62px ~81px ~97px
Gutter width 30px (15px on each side of a column)
Nestable Yes
Offsets Yes
Column ordering Yes

Flex grid based on the CSS flexbox module

For a full explanation on flexbox look here.

A pane grid can be used as a 12 column grid with full height columns. It is also flexible, as it uses a percentage of the viewport to scale the panes.

Define the width by the number of columns, just like the bootstrap grid. For example: .flex-xs-6 will span 6 columns (half of the 12 column grid).

Offset the columns, just like the bootstrap grid. For example: .flex-offset-xs-6 will offset the element 6 columns to the right.

Just like the bootstrap theme the grid has breakpoint variations allowing for different layouts per breakpoint. .flex-xs-*, .flex-sm-*, .flex-md-* and .flex-lg-* To hide a column from a specific breakpoint, use the class .flex-{breakpoint}-hidden.

The following classes are also available:

  • To make a flex row with stretched columns: .flex-row
  • To make a flex column with stretched rows: .flex-column
  • To make a flex child that does not change dimensions when other childs change or get added: .solid-child
  • To make a flex child that changes dimensions as other childs change or get added: .flex-child
  • To fully stretch out a flex child disregarding any other childs: .flex-full
  • To vertically and horizontally center any content in the flex child: .flex-center
