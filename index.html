Introduction

There are two types of components available in the Hippo Theme.

  • CSS components (HTML + CSS)
  • AngularJS components (HTML + CSS + JavaScript)

Note: for AngularJS components, there is also an API reference available.

AngularJS components

There is an API reference available for the AngularJS components and services that are part of the Hippo theme.

Map

<div hippo.theme.map
  markers="points">
</div>

Note: add the script below to your HTML file when you want to use the map directive. 

<script src="http://maps.google.com/maps/api/js?sensor=true"></script>
<script src="http://google-maps-utility-library-v3.googlecode.com/svn/tags/markerclusterer/1.0.2/src/markerclusterer_compiled.js"></script>
Tree
{{ item.title }} 

<div hippo.theme.tree
    items="treeItems"
    callbacks="callbacks"
    selected-item-id="selectedItem.id">
    {{ item.title }}
</div>

Example tree data

$scope.treeItems = [
{
    id: 1,
    title: 'Item 1',
    items: [
        {
            id: 11,
            title: 'Item 1.1'
        }
    ]
}, {
    id: 2,
    title: 'Item 2',
    items: []
}];

Available callbacks

$scope.callbacks = {
    accept: function(data, sourceItemScope, targetScope) {},
    orderChanged: function(scope, sourceItem, sourceIndex, destIndex) {},
    itemRemoved: function(scope, sourceItem, sourceIndex) {},
    itemAdded: function(scope, sourceItem, destIndex) {},
    itemMoved: function(sourceScope, sourceItem, sourceIndex, destScope, destIndex) {},
    itemClicked: function (sourceItem) {}
};

Select Box

<select hippo.theme.select-box
  multiple="multiple" 
  ng-model="selectedValues" 
  ng-options="o.name for o in options" 
  data-placeholder="">
</select>

Note: the options to show must be inside a variable named `options`. If you use an other scope variable name, the list will always be empty.
