map
directive in module hippo.theme

Description

Uses Google Maps to display a map with the markers provided. The markers are grouped when they are close to each other.

Usage

as attribute
<ANY map
     points="{Array}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
pointsArray

A list of point objects, having a longitude- and latitude property, to be visualised. Each having a longitude and latitude property.
