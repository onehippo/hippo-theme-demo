tree
directive in module hippo.theme

Description

Tree component for the Hippo Theme. It uses jsTree to render the tree.

Usage

as attribute
<ANY tree
     data="{object}"
     [on-select="{string}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
dataobject

The data to use for the Tree.

onSelect
(optional)
string

The function to evaluate when a new node in the Tree is selected.
