chart
directive in module hippo.theme

Description

Visualize a list of data with a pie-chart.

Usage

as attribute
<ANY chart
     data="{Array}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
dataArray

A list of data to be visualised, each object of the list having a 'label' and 'value' property.
