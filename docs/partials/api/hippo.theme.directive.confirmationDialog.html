confirmationDialog
directive in module hippo.theme

Description

Renders a confirmation dialog that can show any message and provides a confirm- and cancel button. It has two states, s-visible and s-invisible, which are represented by CSS-classes. The CSS property top will animate when the value is changed in CSS.

Note: the confirmation dialog won't show or hide itself. You can do this easily by assigning the desired CSS properties for the .s-visible and .s-invisible classes in your own CSS.

Usage

as attribute
<ANY confirmation-dialog
     confirm-icon="{string}"
     confirm-label="{string}"
     cancel-label="{string}"
     perform-confirmation="{expression}"
     perform-cancel="{expression}"
     show="{boolean}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails

confirmLabelstring

The label to show for the confirmation button

cancelLabelstring

The label to show for the cancel link

performConfirmationexpression

Function to call when the confirm button is clicked.

Function to call when the cancel button is clicked.
