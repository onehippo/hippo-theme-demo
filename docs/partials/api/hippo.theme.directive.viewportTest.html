viewportTest
directive in module hippo.theme

Description

Detects the current active viewport by creating an empty div-element and attaching Bootstrap 3 classes to it. When the created element is hidden, the related viewport for the class given is set to active.

When the window gets resized, the possible new viewport will automatically be detected and set as active.

Dependencies

  • $window

Usage

as attribute
<ANY viewport-test>
   ...
</ANY>
