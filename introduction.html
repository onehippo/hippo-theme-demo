Introduction

The hippo theme is a css / javascript library based on the Bootstrap framework and includes AngularJS components.

Apart from some base styling of fonts, colors, tables etc it includes 2 grids and 2 types of components. The grid systems are the standard Bootstrap grid system and another grid that uses absolute positioning to provide full height columns as needed in many webapps. The components are divided in:

  • Bootstrap components which are the basic Bootstrap components based on Bootstrap HTML CSS and Javascript.
  • AngularJS components which are Hippo AngularJS directives.

Note: for AngularJS components, there is also an API reference available.
